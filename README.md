### Skincare E-Commerce Data Analysis

## Project Overview
An end-to-end data analysis project on a skincare e-commerce business using Excel, MySQL and Power BI. This project covers the complete data analyst workflow from raw data cleaning to interactive dashboard and business insights.

## Tools Used
Microsoft Excel — Data cleaning and preprocessing
MySQL — Database design, querying and business analysis
Power BI — Interactive dashboard and data visualization

## Dataset Overview
Customers — 500 rows — Customer demographics and acquisition channel
Products — 28 rows — Skincare product details and pricing
Orders — 1,250 rows — Order transactions and delivery status
Order Items — 2,042 rows — Product level order details
Reviews — 494 rows — Customer ratings and feedback
Returns — 79 rows — Return reasons and refund status

## Project Workflow
## Data Cleaning in Excel
Removed duplicate records and blank rows
Standardized date formats to YYYY-MM-DD
Fixed inconsistent column names and values
Handled null values in delivered date column
Prepared clean CSV files for MySQL import

## Database Analysis in MySQL
Designed  orders table relational database for changes a data types
Wrote SQL queries covering monthly sales trends, customer segmentation and profit margins
Used JOINs, Window Functions, CTEs and Subqueries
Calculated Customer Lifetime Value for each customer
Analyzed product profitability comparing revenue against cost price

## Database Structure
The database has 6 related tables including customers, products, orders, order items, reviews and returns tables all linked through customer id, order id and product id columns.
SQL Concepts Used
SELECT, WHERE, GROUP BY, ORDER BY and HAVING
INNER JOIN and LEFT JOIN for multiple table analysis
SUM, COUNT, AVG, MIN, MAX aggregate functions
RANK, ROW NUMBER, LAG, LEAD window functions
Common Table Expressions for complex queries Subqueries
CASE WHEN for customer segmentation
DATEDIFF and DATE functions for time analysis
